# Jogo de advinhação em PYTHON
## Apresentação
- Este projeto consiste em um jogo de advinhação de palavras baseado no famoso jogo TERMOO ou WORDLE.

## Objetivo do jogo:
-  Advinhar a palavra selecionada pelo programa, dentro do limite de tentativas por dificuldade.

## Como Funciona:
1. Baixe o arquivo.py na sua máquina.
2. Execute o arquivo no terminal.
3. Selecione a dificuldade desejada (5,6 ou 7).
    - 5 para palavras de 5 letras, com 5 tentativas. 
    - 6 para palavras de 6 letras, com 6 tentativas.
    - 7 Para palavras de 7 letras, com 7 tentativas.
4. Para inciar novamente, exute novamente pelo terminal.
5. Aproveite

# Requisitos

- Python 3.13
